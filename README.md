# Travel Blog ✈️
This is a travel blog web app built with **React** and **Sanity CMS**.

## 🛠️ Tech Stack
- **React**
- **Sanity.io**
- **Tailwind CSS**
- **Netlify**


## 🚀 Getting Started

# Clone the repo
git clone https://github.com/yourname/travel-blog.git
cd travel-blog

# Install dependencies
npm install

# Start the development server
npm start
